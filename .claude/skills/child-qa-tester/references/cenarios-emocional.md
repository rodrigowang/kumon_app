# Cenários de Teste: Feedback e Segurança Emocional

| Cenário | Comportamento Esperado | Severidade |
|---------|----------------------|------------|
| Primeiro erro do dia | Feedback suave: animação gentil + "Tente de novo!" | **Alta** |
| 3º erro consecutivo | Feedback mais acolhedor: "Você está aprendendo, isso é normal!" + oferecer dica visual | **Crítica** |
| 5º erro consecutivo | Sistema assume que a criança está presa. Regredir nível + mostrar fase concreta como apoio. Tom: "Vamos ver de um jeito diferente!" | **Crítica** |
| Primeiro acerto após sequência de erros | Celebração **maior que o normal** — recompensar a persistência | **Alta** |
| Sequência de 10 acertos | Celebração especial (confete, desbloqueio, som especial). Não deixar a sequência passar batido | **Alta** |
| Criança completa todos os exercícios de uma sessão | Tela de encerramento celebratória. Mostrar o que foi conquistado. Nunca terminar abruptamente | **Alta** |
| Som está desligado no dispositivo | Feedback visual deve ser suficiente sozinho. Testar app inteiro sem som — tudo deve funcionar | **Alta** |
