# Currículo Matemático — Escopo para 7 Anos

O conteúdo segue a progressão natural de uma criança no 2º ano / transição operatório concreto.

## Sequência de Conteúdo (Small Steps)

```
Nível 1: Reconhecimento e escrita de números (0-20)
  → Contar objetos
  → Associar quantidade ao numeral
  → Escrever numerais

Nível 2: Adição simples (resultados até 10)
  → 1+1, 1+2, 1+3... progressão unitária
  → Soma com dedos/objetos visuais (fase concreta)
  → Soma com bolinhas (fase pictórica)
  → Soma apenas numérica (fase abstrata)

Nível 3: Adição com resultados até 20
  → Introdução do "passa 10" (8+5=13)
  → Decomposição visual (8+5 → 8+2+3 → 10+3)

Nível 4: Subtração simples (resultados positivos, até 10)
  → Introdução via "tirar objetos" (concreto)
  → Relação com adição (se 3+2=5, então 5-2=?)

Nível 5: Subtração até 20
  → Inclui "descer do 10" (13-5=8)

Nível 6: Noções de dezena e unidade
  → Agrupamento visual de 10 objetos
  → Representação com barras de dezena + unidades soltas

Nível 7: Adição e subtração com dezenas
  → 10+5, 20+3, 15+10
  → Uso de material dourado visual (barras + cubos)
```

> **Importante**: Esta sequência é um guia. O algoritmo de maestria determina quando a criança avança, não o "nível" em si. Uma criança pode ficar no Nível 2 por semanas até automatizar, e isso é normal e esperado.
